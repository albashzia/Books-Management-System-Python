# Books-Management-System-Python
A simple console-based Books Management System written in Python.  
This project is intended for learning purposes and demonstrates basic data handling, user interaction, and list management in Python.

## Features

- Add books with title, author, and genre
- Search books by genre (case-insensitive)
- Remove books using a keyword from the title
- Menu-driven console interface

## How It Works

The system stores:
- Book titles
- Authors
- Genres
